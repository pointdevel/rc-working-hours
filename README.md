## ✨ Features

- 🌈 Enterprise-class UI designed for web applications.

```bash

```

```bash
yarn add @pointdevel/rc-working-hours
```

## 🔨 Usage

```jsx

const App = () => (
  <>
    <WorkingHours/>
  </>
);
```
